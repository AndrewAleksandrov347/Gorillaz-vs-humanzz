# Gorilla VS Humans

Simple browser game prototype. Open `src/index.html` in your browser to play.

All styles are located in `src/style.css` and the game logic is in `src/game.js`.

## Running the project

If you want to run the prototype locally with Vite:

1. Install dependencies with `npm install`.
2. Start the dev server using `npm run dev`.

This command launches Vite and automatically opens the game in your browser.
